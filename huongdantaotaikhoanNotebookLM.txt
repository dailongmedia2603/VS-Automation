Hướng Dẫn Tích Hợp API Tạo Tài Khoản NotebookLM
Hệ thống đã được nâng cấp để hỗ trợ các phần mềm bên thứ 3 tự động tạo tài khoản và đồng bộ Cookie.

1. 🌐 API Endpoint
URL: https://notebooklm.pcdl.io.vn/api/accounts/add
Method: POST
Content-Type: application/json
2. 📦 Cấu Trúc Dữ Liệu (Payload)
Chỉ cần gửi name và 
cookies
. Trường 
account_id
 là tùy chọn (hệ thống tự sinh nếu bỏ trống).

{
  "name": "Tên Tài Khoản (Ví dụ: User A)",
  "cookies": "__Secure-1PSID=...; __Secure-1PSIDTS=...; ..."
}
Giải Thích:
name (Bắt buộc): Tên hiển thị của tài khoản.
cookies (Bắt buộc): Chuỗi cookie đầy đủ lấy từ trình duyệt (Key=Value; Key=Value).
account_id: (Tùy chọn) Nếu bạn muốn tự quản lý ID (ví dụ: user-001). Nếu không gửi, hệ thống sẽ tự tạo (ví dụ: ten-tai-khoan).
3. ✅ Kết Quả Trả Về (Response)
{
  "success": true,
  "message": "Account \"Tên Tài Khoản\" saved successfully",
  "account_id": "ten-tai-khoan"
}
4. 🔄 Cơ Chế Tự Động
Ngay khi bạn gọi API này:

Tài khoản được lưu vào database.
Cookie được nạp ngay lập tức vào trình duyệt ngầm (Playwright).
Token xác thực (CSRF, SessionID) được làm mới tự động.
Tài khoản sẵn sàng sử dụng (Chat, Upload) ngay lập tức.
5. ⚠️ Hỗ Trợ Endpoint Cũ (Legacy)
Nếu phần mềm của bạn đang dùng endpoint cũ:

URL: https://notebooklm.pcdl.io.vn/api/auth/save
Payload: {"cookies": "..."}
Hệ thống VẪN HỖ TRỢ endpoint này. Nó sẽ tự động tạo một tài khoản tên là Imported Account YYYY-MM-DD HH:MM để đảm bảo dữ liệu không bị mất.
